# Python Practice Questions 🐍

This repository contains a collection of Python programs created while practicing programming logic and problem-solving. The programs were written as part of interview preparation and to strengthen core Python fundamentals.



---

## 📌 About This Repository

The repository includes multiple Python scripts that focus on improving:

* Logical thinking
* Problem solving skills
* Python fundamentals
* Coding interview preparation

Each script represents a separate programming exercise.

---

## 📂 Project Structure

The programs are organized using numbered files:

```
Questions/
│
├── Q1.py
├── Q2.py
├── Q3.py
├── Q4.py
├── Q5.py
└── ...
```

Each file contains the solution for a different programming problem.

---

## 🛠 Technology Used

* Python 3

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/PiyushGaywal/Questions.git
```

2. Go to the project directory

```
cd Questions
```

3. Run any Python file

```
python Q1.py
```
